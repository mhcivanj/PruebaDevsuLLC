EL Patron de Diseño que utilice es SreenPlay(https://dev.to/enriquedcs/patron-de-pruebas-automatizadas-screenplay-2d8e)



1.	Abrir el proyecto con Visual Studio .Net 2022
2.	Ir a la Pestaña Build luego Clean Solution.
3.	Ir a la Pestaña Build luego Build Solution.
4.	Ir a la pestana Test y luego Test Explorer.
5.	Se abrirá en una ventana y cargara los test del proyectos.
6.	Ahora en la ventana de TestExplorer Le damos al botón de play
7.      El proyecto se empezara a ejecutar.


Nota: Ver el documento Word “Ejecutar Proyecto” para mas información.
